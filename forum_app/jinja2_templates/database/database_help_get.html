What is database module (and what is it suppose to do)?

What is this page suppose to do?

Or rather what is the database module?

The database module is suppose to be a place where we apply database scripts.

Here's how it works.

We write scripts that we want to apply to database in app/data/database_scripts folder.
Divided by create table/view/stored proc/functions etc... etc...

On application startup (or manual trigger), we scan the scripts in the folder and note them in applied scripts.
